#### 安装教程

1. clone仓库到你本地
2. 安装flutter运行环境，如果不会的可以关注我的公众号看教程
3. 导入工程到ide，我用的是Vscode
4. 安装依赖flutter packages get 慢的话需要替换国内镜像地址
